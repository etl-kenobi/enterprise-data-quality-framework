The Enterprise Data Quality Framework is a modular, reusable solution designed to validate and enforce data quality during ingestion workflows in modern data platforms.

This project simulates a banking data environment and showcases automated validation using PySpark in Databricks, with integration support for Azure Data Factory (ADF) and CI/CD pipelines.

  • Reusable Validation Module: PySpark class DQValidator supporting:
    • Null Checks
    • Duplicate Checks
    • Range Checks
    • Referential Integrity Checks
  • Batch Validation Notebook: Databricks-compatible PySpark notebook to validate customer and transaction datasets end-to-end.
  • Synthetic Sample Data: Realistic datasets generated using Faker for banking scenarios.
  • ADF CI/CD Automation: GitHub Actions workflow (deploy-adf.yml) for pipeline deployment.
  • Documentation: Includes project-walkthrough.md and use-cases.md
